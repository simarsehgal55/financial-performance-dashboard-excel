# Financial Performance Dashboard (Excel)

## Overview
Automated Excel dashboard to analyse revenue, costs, profitability, and key business KPIs.

## Tools Used
- Excel
- Power Query
- Pivot Tables
- Advanced Formulas

## Key Features
- Automated data cleaning using Power Query
- KPI tracking for revenue, cost, and profit
- Interactive dashboard with filters

## Impact
- Reduced manual reporting effort by ~30%
- Enabled faster financial insights and decision-making
